Organize your WordPress content workflow with custom colored labels, private notes, and backend dashboard widgets.

== Description ==

Label Forge is a WordPress plugin designed to help you organize, manage, and visualize your content workflow directly from the admin area.

Create custom colored labels, assign them to posts or custom post types, add private internal notes, and monitor your content through dashboard widgets with charts, lists, and statistics. Everything stays inside the WordPress backend, making Label Forge ideal for editorial teams, agencies, and anyone managing structured content workflows.

= Main Features =

* **Custom colored labels**: Create labels with custom names, background colors, text colors, border radius, and border styles.
* **Advanced label styling**: Customize borders using solid, dashed, dotted, double, or no border styles.
* **Blinking effect**: Highlight important labels with an optional animated blinking effect between two colors.
* **Private internal notes**: Add backend-only notes to posts and content items for internal organization.
* **Dashboard widgets**: Visualize your content workflow with pie charts, donut charts, bar charts, lists, and stats cards.
* **Interactive charts**: Click chart segments, bars, list items, or stats cards to filter content by label.
* **Quick filters**: Filter content directly from the admin list table using label-based quick filters.
* **Multi-select filtering**: Select multiple labels and filter content using AND or OR logic.
* **Real-time counts**: Label counters update automatically without requiring a full page refresh.
* **Label search**: Quickly find labels through the integrated search field.
* **Drag & drop ordering**: Reorder labels and dashboard widget items with an intuitive drag & drop interface.
* **Clone labels and widgets**: Duplicate existing labels or dashboard widgets with a single click.
* **Default labels**: Customize the built-in “All” and “No Label” labels.
* **Custom Post Type support**: Enable Label Forge for posts, pages, or any public custom post type.
* **Granular permissions**: Control which user roles can view, assign, edit, or manage labels and notes.
* **Backup and restore**: Export and import plugin settings using a JSON file.
* **Smart cache system**: Optimized caching helps keep label counts and dashboard widgets fast.
* **Backend-only workflow**: Labels and notes are visible only in the WordPress admin area.

= Ideal For =

* Editorial teams managing content production.

== Installation ==

1. Upload the `label-forge` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin from the WordPress `Plugins` menu.
3. Go to `Tools > Label Forge` to configure your labels, permissions, and dashboard widgets.
4. Start organizing your content with custom labels and internal notes.

== Frequently Asked Questions ==

= Are labels visible on the frontend? =

No. Labels and notes are visible only in the WordPress backend. They are designed for internal organization and editorial workflow management.

= Can I use Label Forge with Custom Post Types? =

Yes. Label Forge can be enabled for posts, pages, and any supported custom post type from the settings page.

= Can I control who can see or edit labels? =

Yes. Label Forge includes granular role-based permissions, allowing you to control who can view, assign, edit, or manage labels and notes.

= How many labels can I create? =

There is no fixed limit. You can create as many labels as your workflow requires.

= Can I transfer settings between different sites? =

Yes. From the Backup & Restore section, you can export all plugin settings to a JSON file and import them on another site.

= How does multi-select filtering work? =

Multi-select filtering allows you to select multiple labels at the same time. You can choose between AND logic, which shows content matching all selected labels, or OR logic, which shows content matching at least one selected label.

= Does Label Forge affect the frontend of my website? =

No. Label Forge is designed to work only in the WordPress backend and does not display labels or notes on the public frontend.
