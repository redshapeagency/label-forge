# Label Forge

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg) ![WordPress](https://img.shields.io/badge/WordPress-5.0%2B-green.svg) ![PHP](https://img.shields.io/badge/PHP-7.0%2B-8892BF.svg) ![License](https://img.shields.io/badge/License-GPLv2%2B-lightgrey.svg)

Label Forge helps you organize and visualize your WordPress content workflow from the admin area. Create custom colored labels, add private internal notes, and monitor your content through backend-only dashboard widgets with charts, lists and statistics.

---

## Key Features

- Custom colored labels with configurable background, text color, border radius and border style (solid, dashed, dotted, double or none).
- Advanced label styling including optional blinking effect for important labels.
- Private internal notes attached to posts (backend-only).
- Dashboard widgets: pie, donut, bar, list and stats cards with interactive filtering.
- Quick filters in the admin list table with multi-select AND/OR logic.
- Real-time counts and live updates without full page reloads.
- Label search and drag & drop ordering for labels and widget items.
- Clone labels and widgets with one click.
- Granular role-based permissions for viewing, assigning and managing labels and notes.
- Backup and restore plugin settings via JSON export/import.
- Smart cache system to keep counts and widgets responsive.

## Installation

1. Upload the `label-forge` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin from the WordPress **Plugins** menu.
3. Go to **Tools > Label Forge** to configure labels, permissions and dashboard widgets.
4. Start organizing your content with custom labels and internal notes.

## Usage

- Create labels in the Label Forge settings and assign them to posts or custom post types using the meta box.
- Use dashboard widgets and quick filters on the posts list to visualize and filter content.
- Export settings via the Backup & Restore section to move configuration between sites.

## Frequently Asked Questions

Q: Are labels visible on the frontend?

A: No — labels and notes are visible only in the WordPress backend.

Q: Can I use Label Forge with Custom Post Types?

A: Yes — Label Forge supports posts, pages and any public custom post type configurable from the settings.

Q: Can I control who can see or edit labels?

A: Yes — Label Forge implements granular role-based permissions.

Q: How does multi-select filtering work?

A: You can select multiple labels and apply AND logic (content matching all selected labels) or OR logic (content matching any selected label).
